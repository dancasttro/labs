Introdução

O Clickpag é um UI Kit para o projeto Carteira Digital. Ele fornece um guideline de orientações para a estética e interfaces de usuário. Este é um documento vivo e está em constante mudança.

Escrevendo CSS

Para escrever nosso stylesheets nós usamos metodologias e ferramentas que você ter conhecimento. - Usamos um mix para especificar regras como SMACSS, BEM e SUIT para modularizar mais a escrita de componentes. - Nós usamos Stylus como pre-processador em nosso código CSS.
